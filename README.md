Software engineer in Wellington, New Zealand. I focus on debugging, repo rescue, backend systems, DevOps workflows, Python, TypeScript, Docker, GitHub Actions, AWS, API integrations, and production-style data systems.

## Paid debugging / repo rescue

I help developers and small teams get unstuck when a codebase, deployment, or automation workflow is broken.

Send me:

- a GitHub repository link
- the failing command
- the error output or logs
- what you expected to happen
- any setup notes or environment details

I can help with:

- Python traceback/debugging issues
- TypeScript and Node.js backend failures
- Docker and Docker Compose problems
- GitHub Actions / CI failures
- AWS deployment/runtime problems
- API integration bugs
- Playwright scraper and browser automation failures
- broken local development environments

My normal debugging process:

1. Reproduce the problem in a clean environment.
2. Isolate the root cause.
3. Patch the issue in a branch or pull request.
4. Add a test, verification command, or regression note where practical.
5. Document what broke, what changed, and how to prevent it next time.
